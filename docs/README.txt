This plugin enables to integrate GLPI with Piwik

- This plugin allow you to customize you GLPI(http://www.glpi-project.org/) to communicate with Piwik project (http://fr.piwik.org/)

- Tested versions : GLPI v0.83.31, Piwik v1.9.2.

To setup the plugin, you need :
- Install the plugin
- Click on the plugin Piwik, to cutsomize it.
- Fill up information : 
	* site id : Id given by Piwik
	* url : url of your installation of piwik
	* feed Piwik : Yes/No --> Enable feeding or not of Piwik
